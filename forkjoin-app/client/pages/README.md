# Pages

Parent: [../README.md](../README.md)

## Purpose

This directory contains the marketing routes for Aeon Foundation products plus the `/docs` gateway that hands visitors into the canonical docs app.

## Key Files

- `Docs.tsx`: Marketing-to-technical gateway with quickstart, docs, products, library, API, and ebook launch points.
- `AeonPage.tsx`: Product overview that links straight into the Aeon track in `docs-app`.
- `FluxPage.tsx`: Product overview that links straight into the Aeon-Flux track in `docs-app`.
- `DashPage.tsx`: Product overview that links straight into the Dash track in `docs-app`.
- `AegisPage.tsx`: Product overview that links straight into the Aegis track in `docs-app`.
- `DashRelayPage.tsx`: Product overview that links straight into the DashRelay track in `docs-app`.
- `EdgeworkPage.tsx`: Product overview that links straight into the Edgework track in `docs-app`.
- `EcosystemProductPage.tsx`: Shared marketing route for all newly registered ecosystem layers without bespoke pages yet, using the stack registry's narrative copy and docs links.
- `NeuralPage.tsx`: Product overview that links straight into the Neural track in `docs-app`.
