# ForkRaceFoldTheorems

Parent: [Gnosis Lean Modules README](../README.md)

This directory contains standalone theorem files for arithmetic, topology, consensus, and governance claims expressed in Lean.

Selected modules:

- `AeonVoting.lean`: governance-deficit voting theorems for plural community cover, rejection monotonicity, deterministic tie-breaking, and one-stream anti-theorems.
- `GoldenConsensus.lean`: arithmetic threshold theorems for the Fibonacci framing of consensus.
- `MoralLemmas.lean`: arithmetic moral toy theorems.
- `GreekLogic.lean`: arithmetic and paradox toy theorems.
