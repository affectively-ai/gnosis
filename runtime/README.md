# gnosis_runtime

`gnosis_runtime` is the Rust/WASM runtime crate inside the Gnosis project.

The fair brag is that Gnosis is not only a TypeScript toolchain. There is also a lower-level runtime crate aimed at running Gnosis semantics in a compact Rust/WASM form.

## What It Helps You Do

- host runtime logic in Rust
- compile to WASM
- support a lower-level execution layer beneath the broader Gnosis tooling

## Why This README Exists

This crate is a real part of the Gnosis runtime story and should not stay undocumented.
